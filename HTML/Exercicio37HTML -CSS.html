Bootstrap

O Bootstrap é um framework front-end que fornece componentes e ferramentas para facilitar a criação de sites responsivos. Ele foi desenvolvido pelo Twitter e é muito útil para aqueles que precisam de aplicar css a documentos html

Container

Container é uma classe fundamental do Bootstrap, usada para delimitar e alinhar o conteúdo na página, definindo limites para o layout e garantindo o espaçamento adequado entre os elementos.

O seletor de classe container pode ser aplicado de duas formas: container ou container-fluid. Sendo assim podemos dizer que o container quando aplicado delimita larguras fixas nas laterais no documento html. Já o container-fluid ocupa quando aplicado 100% do layout da tela disponível.
