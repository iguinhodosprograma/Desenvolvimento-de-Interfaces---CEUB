Exemplo 23 HTML CSS - Bordas e Alinhamentos

Bordas e alinhamento são um importante recurso em CSS, eles são usados para definir o alinhamento e o posicionamento de elementos HTML.

Podemos utilizar diferentes propriedades para definir a margem (margin), o preenchimento (padding) e a borda (border). Lembrando que a margem e o padding não tem cores, já a border poderá ter uma cor e diferentes formas de espessura e contorno.

Neste exemplo, usamos diferentes aplicações dos três conceitos mencionados. Acesse a página para saber mais.
